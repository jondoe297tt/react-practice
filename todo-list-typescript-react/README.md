## todolist
    这是一个经典的todolis小项目，我用来学习的基本的组件传值。
### 技术栈
+ 使用react 函数式组件 tsx编写主要结构；
+ 使用tailwindcss结合原生css完成样式设计；
+ 使用typescript规范数据类型；
